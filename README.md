## 使用方法

1. 直接下载软件运行

   [dragUpload.dmg](https://github.com/lleohao/dragUpload-qiniu/releases/download/v0.1.1/dragUpload-0.1.1.dmg)

   windows客户端待打包

2. 自行编译运行

   下载项目包

   `git@github.com:lleohao/dragUpload-qiniu.git`

   安转依赖

   `cd dragUpload-qiniu && npm install`

   运行程序

   `npm start`
